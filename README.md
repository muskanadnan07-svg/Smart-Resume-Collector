# Smart Resume Collector using n8n

## Overview

The Smart Resume Collector is an automation workflow built with n8n that collects candidate information through an online form, stores the data in Google Sheets, and automatically sends a confirmation email to each applicant.

## Features

* Resume application form
* Automatic data storage in Google Sheets
* Confirmation email using Gmail
* No-code workflow automation
* Easy to customize for recruitment processes

## Technologies Used

* n8n
* Google Sheets
* Gmail API
* Google OAuth

## Workflow

Form Submission
↓
Google Sheets
↓
Gmail Confirmation

## Future Improvements
